# Xiaxue Ouyang Homepage

This is the source code for:

`https://xiaxueouyang2001.github.io`

## Publish with GitHub Pages

1. Rename the GitHub account to `xiaxueouyang2001`.
2. Create a public repository named `xiaxueouyang2001.github.io`.
3. Upload `index.html`, `styles.css`, `script.js`, and the `assets` folder.
4. Wait a few minutes, then open `https://xiaxueouyang2001.github.io`.

## Update the homepage later

Edit `index.html`, upload the new version to the same repository, and replace the old file.

If you add a CV, save it as `assets/cv.pdf` and change the CV link in `index.html` from `assets/cv-placeholder.txt` to `assets/cv.pdf`.
